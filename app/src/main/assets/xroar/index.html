About

This Android app packages the XRoar emulator with a touch-friendly interface.

XRoar is the Dragon and Tandy 8-bit computer emulator project by Ciaran Anscomb.

Credit and thanks go to the original XRoar project for the emulator core used here.

Official project site: https://www.6809.org.uk/xroar/

Info

Quick start help for Android XRoar.

1. First import your own ROM files. Open Config and use Import ROM Folder. XRoar needs your firmware ROMs before most machines will boot.

2. Return to the Screen page after the ROM import reload finishes. The emulator display is the main home page.

3. Use Media to load software. You can insert disks, tapes, or use Quick Load for files that XRoar can auto-detect.

4. Use the on-screen keyboard or gamepad on the home screen depending on what the software expects.

5. If needed, use Soft Reset or Hard Reset from the top bar to restart the emulated machine.
